Smart Calc v1.0

About

  • This program evaluates arithmetic expressions in infix notation. Also Smart Calc can build graphs.

How to install:

1) make install
2) Launch Smart Calc from Desktop icon

Information to use:

1) Use it like basic calculator.
2) Button plot graph to open graph plotter window.
3) use brackets for unary operations (-a) (+b)
4) Instead of empty values, default values are entered.

Requirements:

1) QT
2) qmake
3) check.h & lcov for tests
4) gcc
