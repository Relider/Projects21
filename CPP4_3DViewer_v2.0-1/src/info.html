3d Viewer v1.0

About

  • This program opens obj files for viewing. Affine transformations are used to change the position of an object. The OpenGl library is used for graphic rendering.

How to install:

1) make install
2) Launch 3d Viewer from Desktop icon

Information to use:

1) Use it to open obj files.
2) Button Open file to open obj file.
